# Banque IA

Plateforme autonome et sécurisée de diffusion d'informations et de copilote intelligent pour une banque.

## Blocs de la V1
- Admin web
- Portail interne employés
- Portail externe public
- Backend API sécurisé

## Principes
- Aucun accès aux comptes bancaires
- Aucun accès aux transactions
- Aucun lien avec le core banking
- Réponses uniquement à partir des documents validés
- Gestion stricte des rôles et permissions

## Lancement backend
```bash
cd backend
npm install
npm run dev
